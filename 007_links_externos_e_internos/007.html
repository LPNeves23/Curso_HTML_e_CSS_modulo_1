Usando links externos e internos

Link externo

Para levar o usuário a um link externo, basta usar a tag a. Ex:
Conheça meu GitHub.

para que, ao clicar no link, o usuário não saia do site, você utiliza as tags target="blank" e rel="external" logo em seguida. Veja ex abaixo:
Conheça meu GitHub.

Link Interno

Para navgar por um link interno, basta usar os mesmos comandos do externo, adicinando o endereço de sua página interna ao invés do link externo. Veja exemplo abaixo:
Clique AQUIpara ir a segunda página.

Links para download

Para realizar downloads, basta você utilizar a tag a assim como nos links. e também não se esquecer de colocar a frente a tag download com o nome do arquivo e também com a tag type a frente para especificar o tipo de arquivo que será baixado. Veja o exemplo abaixo:

Baixe aqui meu livro.

Baixe aqui o arquivo zip de meu livro

OBS: Para descobrir o type do arquivo que será baixado, acesse o site IANA e procure pelo formato
