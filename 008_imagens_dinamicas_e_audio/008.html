Imagens dinâmicas

Precisamos adaptar nossas imagens para que ela fique em um tamanho bom para as várias formas que o usuário pode acessar.

Basicamente, precisamos de 3 imagens:

  • Grande - Para usuários que irão acessar pelo computador;
  • Média - Para usuários que irão acessar de tablets;
  • Pequena - Para usuários que irao acessar de smartphones.

Como adicionar Imagens Dinâmicas

Para adicionar várias imagens para carregar de acordo com o tamanho da tela do usuário, usaremos as tags picture e source.

OBS: Sempre trabalhar com o tamanho máximo e sempre a imagem menor acima da maior. Olhar o código-fonte para qualquer dúvida.
